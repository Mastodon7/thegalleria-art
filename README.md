# The Galleria — Carolyn Elaine

Ready for deployment in Coolify.

## Coolify
1. Push this folder to a GitHub repository.
2. In Coolify choose Public Repository or Private Repository.
3. Build Pack: Dockerfile.
4. Port: 80.
5. Add domains:
   - https://thegalleria.art
   - https://www.thegalleria.art
6. Deploy.

The portfolio is available at the root domain.
